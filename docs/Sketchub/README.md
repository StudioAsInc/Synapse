## About Synapse 

Synapse is an open-source social media application developed using Sketchware Pro. It serves as a foundational template, providing a practical learning resource for developers. By exploring its structure, you can experiment with its features and adapt it for your own projects. This makes it an ideal starting point for those looking to understand the mechanics of a functional social media app.

You can now use Synapse directly through the website, eliminating the need to download the app.

## Installation and Setup

### Prerequisites

*   A working installation of [Sketchware Pro (Nightly)](https://sketchware.pro/download).
*   The `SynapseSketchubShare v1.0.0 SNAPSHOT (com.synapse.social.studioasinc, 40011) 2025-07-19T162856.swb` project file. [Download the project](https://web.sketchub.in/p/28085).

*   Your own credentials for **Firebase, Imgbb, and Cloudinary.**

### Configuration

To get the app fully operational, you'll need to integrate your own credentials for the following services:

1.  **Firebase:** For backend services like authentication, database, and storage.
2.  **Imgbb:** For image hosting.
3.  **Cloudinary:** For optimized image and video management.

These credentials need to be added to the following Java files within the project:

*   `UploadImage.java` (for Imgbb)
*   `FasterCloudinary.java` (for Cloudinary api)

After updating these files with your credentials, the project should be ready to compile.
