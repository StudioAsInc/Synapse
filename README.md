## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- npm (v8+) or yarn
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/StudioAsInc/Synapse.git
cd Synapse

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local

# Start development server
npm run dev
